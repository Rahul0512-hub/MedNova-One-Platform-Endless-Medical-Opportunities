# MedNova

MedNova is a platform that brings together medical learning, career growth, research opportunities, internships, scholarships, events, and networking opportunities in one place.

## Features

* Opportunity Discovery
* Learning Resources
* Events & Workshops
* Career Development
* Student Networking
* Responsive UI

## Tech Stack

* React.js
* Vite
* JavaScript
* HTML/CSS

## Installation

```bash
git clone https://github.com/Rahul0512-hub/MedNova-One-Platform-Endless-Medical-Opportunities.git
cd MedNova-One-Platform-Endless-Medical-Opportunities
npm install
npm run dev
```
